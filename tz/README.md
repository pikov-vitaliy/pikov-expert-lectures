# tz.pikov.expert

Авторский учебный лендинг-презентация Виталия Александровича Пикова по написанию технических заданий для ПО, автоматизированных систем и проектов безопасной разработки.

## Формат

Текущая версия сделана в стиле лендинга `risk.pikov.expert`:

- `104` крупных учебных экрана;
- режим `Слайды` для проведения лекции на проекторе;
- режим `Лонгрид` для самостоятельного чтения слушателями;
- обзор всех слайдов сеткой;
- переключение размера шрифта `A / A+ / A++`;
- светлая и темная тема;
- навигация клавиатурой, кнопками, колесом мыши и deep links `#slide-N`;
- скачивание основной методички в формате Markdown.

По умолчанию включен режим `A+`: базовый текст 24px, крупные заголовки и читаемые красные акцентные блоки для лекционных остановок.

## Состав

```text
tz-pikov-expert/
├── index.html                         # автономный лендинг-презентация
├── README.md
├── .htaccess
├── robots.txt
├── sitemap.xml
├── tools/
│   └── build-tz-landing.mjs           # генератор index.html на 104 слайда
├── assets/
│   ├── css/style.css                  # legacy, текущий index.html использует inline CSS
│   ├── js/main.js                     # legacy, текущий index.html использует inline JS
│   └── images/
│       ├── favicon.svg
│       ├── logo-mascom.svg
│       └── logo-mascom.png
└── materials/
    ├── лекции/
    │   └── методичка-по-написанию-ТЗ.md
    ├── шаблоны/
    ├── примеры/
    └── дополнительно/
```

`photo.jpg` может оставаться в assets как неиспользуемый файл, но новый лендинг его не подключает.

## Пересборка

Если нужно изменить раскадровку или текст слайдов:

```powershell
node .\tools\build-tz-landing.mjs
```

Генератор проверяет, что итоговый лендинг содержит ровно `104` слайда.

## Управление

| Действие | Управление |
|---|---|
| Следующий / предыдущий слайд | `→`, `←`, `PageDown`, `PageUp`, кнопки снизу |
| Начало / конец | `Home`, `End` |
| Обзор всех слайдов | `Esc` или кнопка сетки |
| Слайды / лонгрид | `G` или кнопки в шапке |
| Размер шрифта | `A` или кнопка `A+` |
| Светлая / темная тема | `T` или кнопка темы |
| Полный экран | `F` или кнопка fullscreen |

## Публикация

Загрузить содержимое папки `tz-pikov-expert/` в корень поддомена `tz.pikov.expert`.

Минимальный состав для публикации:

- `index.html`;
- `.htaccess`;
- `robots.txt`;
- `sitemap.xml`;
- `assets/images/logo-mascom.png`;
- `assets/images/favicon.svg`;
- вся папка `materials/`.

Файл `tools/build-tz-landing.mjs` на хостинг загружать не обязательно, но он полезен как исходник для дальнейшего развития курса.

## Проверки перед загрузкой

```powershell
node .\tools\build-tz-landing.mjs
Select-String -LiteralPath .\index.html -Pattern 'src="https://|href="https://fonts|jsdelivr|gstatic|mermaid'
```

Ожидаемо:

- 104 слайда;
- нет подключения `photo.jpg`;
- нет runtime-зависимостей от внешних CDN;
- ссылка `Markdown` ведет на `materials/лекции/методичка-по-написанию-ТЗ.md`;
- локальные ссылки существуют.

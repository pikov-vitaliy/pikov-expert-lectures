# risk.pikov.expert

Папка содержит готовый авторский лендинг по дню I программы профессиональной
переподготовки УЦБИ МАСКОМ: риски информационной безопасности, безопасность
значимых объектов КИИ и реагирование на инциденты.

## Состав

| Файл | Назначение |
|---|---|
| `index.html` | Автономная веб-лекция в формате слайдов и лонгрида. |
| `materials.md` | Полные раздаточные материалы для слушателей и преподавателя. |
| `assets/mascom-logo.png` | Логотип, используемый в оформлении. |
| `.htaccess`, `robots.txt`, `sitemap.xml` | Служебные файлы публикации поддомена. |

## Публикация

Публикуемый минимум для `https://risk.pikov.expert/`:

- `index.html`;
- `materials.md`;
- `assets/mascom-logo.png`;
- `.htaccess`, `robots.txt`, `sitemap.xml`.

- Актуальность нормативной базы, особенно 187-ФЗ, приказов ФСТЭК и ФСБ,
  отраслевых правил категорирования и методик оценки критичности уязвимостей.
- Корректность ссылок на `materials.md` и соседние лекции.
- Единые публичные контакты: `vitaly@pikov.expert`, `@UnderLineSecurity`,

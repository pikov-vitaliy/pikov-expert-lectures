# Notices and Third-Party Material

This repository is prepared as an educational website source repository. It may reference or contain third-party names, trademarks, documents, standards, product materials, screenshots, logos, and generated reference data.

## General Rules

- Third-party trademarks and product names belong to their respective owners.
- Third-party documents keep their original copyright and license terms.
- If a document is publicly available from an authoritative source, prefer linking to that source instead of redistributing a copy when rights are unclear.
- If a file cannot be confidently redistributed, keep it out of the public Git history.

## Known Review Areas

Before the repository is made public, review these areas carefully:

- Astra Linux documentation and PDF materials;
- vendor and course handouts in PDF/PPTX/DOCX format;
- logos in `is/logo` and course-branded images;
- SPDX-derived website/data under `spdx`;
- any copied screenshots or generated snapshots from previously published pages.

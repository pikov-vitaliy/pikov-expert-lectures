# threats-kii.pikov.expert

Папка содержит готовый авторский лендинг по дню II программы профессиональной
переподготовки УЦБИ МАСКОМ: объекты КИИ, угрозы, меры защиты и уязвимости.

## Состав

| Файл | Назначение |
|---|---|
| `index.html` | Автономная веб-лекция в формате слайдов и лонгрида. |
| `materials.md` | Полные раздаточные материалы для слушателей и преподавателя. |
| `assets/mascom-logo.png` | Логотип, используемый в оформлении. |
| `.htaccess`, `robots.txt`, `sitemap.xml` | Служебные файлы публикации поддомена. |

## Публикация

Публикуемый минимум для `https://threats-kii.pikov.expert/`:

- `index.html`;
- `materials.md`;
- `assets/mascom-logo.png`;
- `.htaccess`, `robots.txt`, `sitemap.xml`.

## Связь с днём I

Эта лекция продолжает [risk.pikov.expert](https://risk.pikov.expert/). День II
опирается на риск-методологию дня I и использует её как основу для разбора
угроз, мер защиты и уязвимостей значимых объектов КИИ.

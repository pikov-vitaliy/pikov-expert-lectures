# threats-kii.pikov.expert — комплект для публикации

Папка содержит исходники для подготовки авторского лендинга по **Дню II** (Объекты КИИ: угрозы, меры, уязвимости) программы профессиональной переподготовки УЦ МАСКОМ. Лендинг разместится на поддомене **`https://threats-kii.pikov.expert/`**.

## Состав

| Файл | Что это | Кому |
|------|---------|------|
| `Лекция Угрозы КИИ - материалы.md` | Большой markdown (~140 КБ) с полным содержанием 5-блочной лекции (5 × 1,5 ак. ч). Структурирован по расписанию дня, с разделителями блоков, перерывами и **практическим заданием** в блоке 4. | Слушателям — раздаточный материал; преподавателю — исходник для презентации и актуализации; Claude Designer — источник содержания. |
| `materials.md` | Копия markdown под коротким именем для удобной публичной ссылки на лендинге. | Публикуется на хостинге, ссылка из финального слайда. |
| `Промпт для Claude Designer.md` | Готовый промпт-ТЗ для Claude: создать `index.html` лендинга (~110–115 слайдов) с раскадровкой по 5 блокам. Точная копия дизайна `risk.pikov.expert`. | Тому, кто будет генерировать `index.html` через Claude. |
| `robots.txt` | Allow + ссылка на sitemap. | Хостинг. |
| `sitemap.xml` | 1-URL sitemap для поисковиков. | Хостинг. |
| `.htaccess` | UTF-8 charset для .md/.txt + защита .htaccess. | Хостинг. |
| `README.md` | Этот файл. | — |

## Как использовать

1. **Получить `index.html`** через Claude:
   - Откройте Claude (chat или Claude Code).
   - Вставьте содержимое `Промпт для Claude Designer.md`.
   - В отмеченное место в промпте вставьте содержимое `Лекция Угрозы КИИ - материалы.md`.
   - Дополнительно приложите `risk/index.html` как стилистический референс.
   - Получите `index.html`. Сохраните рядом с этим README в папке `threats-kii/`.

2. **Загрузите на хостинг** в корень поддомена `threats-kii.pikov.expert`:
   - `index.html` (главный лендинг);
   - `materials.md` (для ссылки «Материалы лекции для прочтения» с финального слайда);
   - `sitemap.xml`, `robots.txt`, `.htaccess`;
   - `assets/mascom-logo.png` (если используется в дизайне).

3. **Обновите корневой каталог** `pikov.expert`:
   - В `index.html` корня добавьте карточку для `threats-kii.pikov.expert` (по аналогии с другими дисциплинами «Регулирование и право»);
   - В корневом `sitemap.xml` добавьте URL `https://threats-kii.pikov.expert/`;
   - В корневом `robots.txt` добавьте `Sitemap: https://threats-kii.pikov.expert/sitemap.xml`;
   - Обновите счётчик курсов и `numberOfItems` в JSON-LD.

## Связь с днём I

Этот день — продолжение [risk.pikov.expert](https://risk.pikov.expert/) (день I). В лекции день II опирается на риск-методологию дня I (краткий рекап на слайде 2 + ссылка). Слушателям, которые не были на дне I — рекомендуется хотя бы пролистать лендинг risk перед лекцией.

## Что обязательно проверить вручную перед публикацией

1. **Актуальность нормативки** на момент проведения лекции:
   - Свежие отраслевые ПП 2026 (атомная энергия, банки, наука, недвижимость, ракетно-космическая, связь);
   - Пакет приказов ФСБ декабря 2025 (№ 539, 546, 547, 548, 553, 554);
   - Проект изменений приказов 235/239 от 07.04.2026.
2. **Дата** проведения лекции (в шаблоне указан четверг 28.05.2026).
3. **Соответствие фирменному стилю** — копия дизайна risk.pikov.expert.
